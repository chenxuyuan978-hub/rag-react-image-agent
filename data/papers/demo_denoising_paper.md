# Image Denoising Experiment Based on Gaussian Blur

## Abstract

Image denoising is a fundamental task in image processing. The goal of image denoising is to remove noise from a degraded input image while preserving important image structures such as edges and textures. In this experiment, Gaussian blur is used as a classical baseline method for image denoising.

## Experiment Setting

The input image is a noisy RGB image generated by adding random Gaussian noise to a clean reference image. The reference image is used as the ground truth for quantitative evaluation.

The denoising method used in this experiment is Gaussian blur. The Gaussian blur kernel size is set to 5. This setting is chosen because a small kernel can suppress image noise while avoiding excessive loss of image details.

## Method

Gaussian blur smooths an image by applying a Gaussian kernel to local neighborhoods. For each pixel, the output value is computed as a weighted average of surrounding pixels. Pixels near the center of the kernel have larger weights, while pixels farther away have smaller weights.

In this experiment, the following processing pipeline is used:

1. Load the noisy input image.
2. Apply Gaussian blur with kernel size 5.
3. Optionally apply image sharpening to enhance local structures.
4. Compare the processed image with the clean reference image.

## Evaluation Metrics

The experiment uses three common image quality evaluation metrics:

- MSE: Mean Squared Error. A lower MSE indicates smaller pixel-level difference.
- PSNR: Peak Signal-to-Noise Ratio. A higher PSNR indicates better reconstruction quality.
- SSIM: Structural Similarity Index. SSIM measures structural similarity between two images, and a value closer to 1 indicates higher similarity.

## Expected Result

After Gaussian blur processing, random noise in the input image should be reduced. However, excessive smoothing may also blur edges and fine details. Therefore, the experiment result should be analyzed by considering both visual quality and quantitative metrics such as MSE, PSNR, and SSIM.

## Conclusion

Gaussian blur is a simple and effective baseline method for image denoising. Although it cannot preserve fine details as well as advanced learning-based methods, it provides a clear and reproducible baseline for image processing experiments.
