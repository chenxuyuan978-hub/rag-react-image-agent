# Demo Image Processing Paper

## Abstract

This demo paper studies image denoising with a simple local image processing
pipeline. The method uses Gaussian blur as a baseline filter for noisy images.

## Experiment Setting

The experiment setting uses a synthetic RGB image and a noisy input image. We
apply Gaussian blur with kernel size 5 before measuring image quality.

## Evaluation

The reproduced experiment reports MSE, PSNR, and SSIM. MSE measures pixel error,
PSNR measures signal quality, and SSIM measures structural similarity.
